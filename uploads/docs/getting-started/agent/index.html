    Agent管理¶

    Agent介绍¶

    1.Agent包存放位置。

    Agent的包存放在/opt/opsany/uploads/agent目录下，该目录再OpenResty上设置了可以下载

    cd /opt/opsany/conf/nginx-conf.d
cat nginx_paas.conf
...
    location ~ ^/uploads/(.*) {
        autoindex off;
        root /opt/opsany/;
    }
...

    2.Agent包构成

    Agent是集成了salt-minion、zabbix-agent、filebeat的综合包，使用supervisor进行进程管理，免安装直接解压即可，但是因为salt-minion依赖于Python3，所以在安装脚本中会先安装Python3 ，如果安装失败，需要手工修改EPEL的地址。

    vim /opt/opsany/uploads/agent/agent-install.sh +14
EPEL6="http://mirrors.aliyun.com/repo/epel-6.repo"
EPEL7="http://mirrors.aliyun.com/repo/epel-7.repo"

    Agent管理¶

    1.查看Agent状态

    systemctl status opsany-agent

    2.查看Agent中插件状态

    /usr/local/opsany-agent/agent.sh status

    3.重启所有Agent服务

    /usr/local/opsany-agent/agent.sh restart all

    4.查看Agent日志

    cd /usr/local/opsany-agent/log
